Política de Privacidade

Aplicativo: N8n-TCC   |   Uso: Testes e uso pessoal

Esta Política de Privacidade descreve como o aplicativo N8n-TCC coleta, usa, armazena e protege dados quando utilizado para integrações e automações entre plataformas (por exemplo: Instagram, Facebook, Telegram) por meio da ferramenta n8n.

1. Dados que podemos coletar

Dependendo do uso do aplicativo e das integrações habilitadas, podemos coletar os seguintes tipos de informações:

  • Dados fornecidos pelo usuário: mensagens de chat, nome de usuário, número de telefone ou outros dados que o usuário envie voluntariamente nos canais integrados.
  • Dados técnicos: identificadores de sessão, informações de conexão (IP, hora, user-agent) e logs de sistema necessários para operar os webhooks e integrações.

2. Finalidade do tratamento

Os dados coletados são utilizados exclusivamente para:

  • Habilitar o funcionamento das automações e chatbots entre plataformas (encaminhar mensagens, enviar respostas automáticas, acionar fluxos no n8n).
  • Diagnóstico de erros, monitoração e melhoria dos fluxos enquanto o app estiver em uso para testes pessoais.
  • Cumprir solicitações do usuário relacionadas ao próprio uso (ex.: exportar ou excluir conversas armazenadas temporariamente).

3. Compartilhamento de dados

O aplicativo N8n-TCC não vende, aluga ou troca dados pessoais com terceiros para fins comerciais. Os dados podem ser encaminhados a provedores e APIs necessárias para executar integrações (por exemplo, a API do Instagram/Facebook ou o Telegram) — nesses casos os dados trafegam conforme as políticas dessas plataformas.

4. Armazenamento e retenção

Por se tratar de uma implementação destinada a testes e uso pessoal, o armazenamento é limitado ao necessário para a execução dos fluxos. Caso você configure armazenamento persistente (banco de dados, planilhas, arquivos), revise onde esses dados ficam armazenados. Recomendamos:

  • Manter logs por um período mínimo necessário e removê-los manualmente após os testes.
  • Proteger acesso ao painel n8n com autenticação forte.

5. Segurança

Adotamos medidas razoáveis de segurança técnica e administrativa para proteger os dados contra acesso não autorizado, perda ou alteração. No entanto, por se tratar de ambiente de testes, recomenda-se cautela ao publicar dados sensíveis.

6. Direitos do titular dos dados

Você tem direito a solicitar:

  • Acesso às informações pessoais vinculadas às automações;
  • Correção de dados incorretos;
  • Solicitação de exclusão dos dados coletados (quando aplicável).
